- 在 Query 方法中添加 `context.WithTimeout(ctx, c.timeout)`

### 5.2 实现指数退避重试
```go
func (c *Client) queryWithRetry(ctx context.Context, query string) (model.Value, error) {
    var lastErr error
    for i := 0; i <= c.maxRetries; i++ {
        result, err := c.doQuery(ctx, query)
        if err == nil { return result, nil }
        lastErr = err
        if !isRetryable(err) { break }
        time.Sleep(c.retryDelay * time.Duration(1<<uint(i)))
    }
    return nil, lastErr
}
```
使用 `PrometheusConfig` 中已有的 Timeout / MaxRetries / RetryDelay 字段。

### 5.3 NewClient 返回 error 而非 panic
签名: `func NewClient(cfg config.PrometheusConfig) (*Client, error)`

### 5.4 更新 Wire Provider
